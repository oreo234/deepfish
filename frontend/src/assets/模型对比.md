## 二、指标说明

| 指标 | 全称 | 含义说明 | 结论项 | 总结 |
| --- | --- | --- | --- | --- |
| 指标 | 全称 | 含义说明 |  |  |
| mAP@0.5 | Mean Average Precision at IoU = 0.5 | 在交并比 (IoU) ≥ 0.5 时，所有类别的平均检测精度均值。反映检测框位置准确性。数值越高越好。 | 结论项 | 总结 |
| mAP@0.5:0.95 | Mean Average Precision averaged over IoU thresholds (0.5–0.95) | 在 IoU 0.5~0.95（步长0.05）的区间上平均的 mAP，衡量模型对不同精度要求下的综合检测能力，更严格、更全面。 | 精度表现最优 | YOLO（mAP@0.5 = 0.85）整体检测精度最高。 |
| Precision（精确率） | TP / (TP + FP) | 表示预测为正的样本中有多少是真正的正样本。高精度意味着误报少。 | 召回率最高 | Faster R-CNN（Recall = 0.8895）检测到的目标最多，但速度最慢。 |
| Recall（召回率） | TP / (TP + FN) | 表示所有正样本中有多少被正确检测出来。高召回意味着漏检少。 | 速度最快 | YOLO（1分15秒完成测试），效率显著高于其他两种模型。 |
| F1-Score | 2×(Precision×Recall)/(Precision+Recall) | 综合平衡 Precision 与 Recall 的调和平均数，用于整体性能评估。 | 平衡性较好 | YOLO 在精度、召回与速度上表现均衡。 |
| Box Loss | Bounding Box Loss | 表示模型在预测边框位置时的误差，值越小说明框定位更准确。 |  |  |
| Cls Loss | Classification Loss | 表示模型在分类（识别类别）时的损失，值越小说明分类越准确。 |  |  |
| 用时 | Evaluation Time | 模型在测试集上完成一次完整评估所需的时间，格式为分钟:秒。越短说明推理效率越高。 |  |  |
